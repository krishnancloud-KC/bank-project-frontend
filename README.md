# 🏦 GCP Bank App — Frontend

> React Banking Dashboard deployed on GCP Cloud Run

## Features
| Feature | Description |
|---------|-------------|
| Authentication | JWT-based login & registration |
| Dashboard | Real-time account balance & summary |
| Transactions | Deposit, withdraw & history view |
| AI Chatbot | Vertex AI powered banking assistant |
| Fraud Alerts | Real-time suspicious activity notifications |

## Tech Stack
`React.js` `GCP Cloud Run` `JWT Auth` `Docker` `GitHub Actions`
